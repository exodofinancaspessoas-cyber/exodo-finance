
# Êxodo Finance - Personal Finance Management App

## Sobre o Projeto
O **Êxodo Finance** é uma aplicação web moderna de gestão financeira pessoal, desenvolvida para ajudar usuários a alcançar a liberdade financeira através de controle, planejamento e inteligência de dados.

O projeto foi construído com foco em **UX/UI Premium**, performance e funcionalidades avançadas de análise financeira.

## Tecnologias Utilizadas
*   **React** (Vite + TypeScript)
*   **Tailwind CSS** (Estilização avançada e responsiva)
*   **Lucide React** (Ícones modernos)
*   **Recharts** (Gráficos e visualização de dados)
*   **Local Storage** (Persistência de dados dados para o MVP)

## Funcionalidades Principais (MVP)

### 1. Dashboard Inteligente
*   Visão geral do saldo total (soma de todas as contas).
*   Resumo de Receitas vs Despesas do mês atual.
*   Gráfico de "Maiores Gastos" por categoria.
*   Atividade recente com identificação visual rápida (ícones e cores).
*   **AI Insight**: Dicas automáticas baseadas no comportamento financeiro do mês.

### 2. Gestão de Transações Avançada
*   CRUD completo (Criar, Ler, Atualizar, Deletar).
*   **Busca em Tempo Real**: Pesquise por descrição ou observação.
*   **Filtros Combinados**: Data, Categoria, Tipo, Status, Conta, Valor.
*   **Ações em Massa**: Exclua múltiplas transações de uma vez.
*   **Exportação de Dados**: Gere relatórios em CSV, JSON ou PDF/Impressão com colunas customizáveis.

### 3. Relatórios e Análise
*   Gráfico de evolução de gastos mensais.
*   Indicadores de saúde financeira (Média de gastos, Consistência).
*   **Previsões (AI)**: Estimativa de gastos futuros com base no histórico.
*   **Sugestões de Economia**: Algoritmo que identifica categorias com aumento de gastos.

### 4. Gestão de Contas e Cartões
*   Gerenciamento de contas bancárias (Corrente, Poupança, etc).
*   Controle de cartões de crédito com dia de fechamento e vencimento.
*   Cálculo automático de saldo.

### 5. Categorias e Metas
*   Criação de categorias personalizadas com cores e ícones.
*   Definição de orçamentos (budgets) por categoria.
*   Acompanhamento de metas financeiras (ex: "Viagem", "Reserva de Emergência").

## Como Rodar o Projeto

1.  **Instalar dependências**:
    ```bash
    npm install
    ```

2.  **Rodar servidor de desenvolvimento**:
    ```bash
    npm run dev
    ```

3.  **Build para produção**:
    ```bash
    npm run build
    ```
